## Rules
1. Component names: PascalCase

3. All API calls via services/ — NEVER fetch directly in components
4. Tailwind only — NO inline styles, NO CSS modules
5. Every page must be responsive (mobile-first)
6. Dark mode: every component needs dark: variant
7. i18n: all text via translation keys, NO hardcoded Vietnamese in JSX
8. Types: strict TypeScript, no `any`
9. API contract: see api-contracts/ in backend repo
